Data Store Manager APPlication

Application Jar will be available under this location: DataStoreManager/out/artifacts/DataStoreManager_jar/
Download link page : https://github.com/sbsathish999/DataStoreManager/blob/master/out/artifacts/DataStoreManager_jar/DataStoreManager.jar


How to Run:
1. Download jar file.
2. open command promt/ terminal 
3. move to the jar file location
4. Enter this command without quotes "java -jar DataStoreManager.jar"

Note: The code in the repository has been developed with IntelliJ platform.
